## Installation

```bash
python -m pip install --upgrade git+https://codeberg.org/mentalblood/continental
```

## Usage

### Create

```bash
cat ~/Downloads/telegram_dumps/*.json | python -m continental adapt -m telegram -c '{"users": ["–°—Ç–µ–ø–∞–Ω –ù–µ—á–µ–ø–æ—Ä–µ–Ω–∫–æ"]}' | python -m continental create -o ~/continental_nets/–°—Ç–µ–ø–∞–Ω\ –ù–µ—á–µ–ø–æ—Ä–µ–Ω–∫–æ
```

### Generate

```bash
python -m continental generate -i ~/continental_nets/–°—Ç–µ–ø–∞–Ω\ –ù–µ—á–µ–ø–æ—Ä–µ–Ω–∫–æ -l 1000
```
